Objetivos da Disciplina

Desenvolvimento de sistema microcontrolado para controle de temperatura e armazenamento de dados em tempo real, visando a estabilidade do processo de produção da cerveja Witbier.

ControladorESP32 / Arduino
SensorDS18B20 (Waterproof)
AtuadorMódulo Relé
ProtocoloHTTP / Flask API

Fluxograma do Processo Cervejeiro

FASE QUENTE: Moagem ➔ Mostura (62-72°C) ➔ Clarificação ➔ Fervura (95-100°C)
⬇
FASE FRIA: Resfriamento ➔ Aeração ➔ Fermentação (18-24°C) ➔ Maturação ➔ Envase

*Fase Quente: ~8 horas de processo. Fase Fria: Elevado risco de contaminação, exige sanitização rigorosa.
